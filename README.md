
#  Alpha Chess Master PRO (vs AI)

This is a simple chess game I built in Python where you can play against an AI.  
The AI plays as Black, and you play as White.

I made this project to learn how chess games and basic AI actually work in real life.

---

# What you can do

- Play chess against the computer
- Move pieces by dragging and dropping
- See legal moves while playing
- AI automatically responds after your move
- Highlights last move and shows check situation
- Pawn promotion (auto-queen)
- Sound effects for moves and checks
- Shows "AI Thinking..." while AI is calculating
- Game ends automatically on checkmate


# How the AI works

The AI is not random — it actually thinks before moving.

It uses:
- Minimax algorithm
- Alpha-beta pruning (to make it faster)
- Simple piece value system (like queen > rook > pawn)

I built this project to understand:

How chess rules are implemented in code
How AI makes decisions step by step
How games are built using Python
